# 📊 Income Inequality Analysis Dashboard

## Project Overview

This project explores a critical real-world question:

> **How much income is controlled by the top 5% across countries over time?**

What started as a simple data analysis project turned into a powerful insight into global wealth distribution.

---

## 🌍 Problem Statement

Income inequality is one of the most important economic challenges today.

This project analyzes:

- How wealth is distributed among the top 5%
- How this distribution changes over time
- Differences between countries
- Sudden spikes and trends in inequality

---

## 📈 Key Insights

- The income gap is **consistently increasing**
- Some countries show **stable inequality patterns**
- Others show **sharp spikes** (possibly due to policies or economic events)
- Over time, the **top 5% gains more control**

---

## 🛠️ Tech Stack

- **Python**
- **Pandas** → Data Cleaning & Processing  
- **Plotly** → Interactive Visualizations  
- **Streamlit** → Dashboard UI  

---

## 💻 Features

- 🌍 Country-wise comparison  
- 📅 Time range selection (1946–2015)  
- 📊 Interactive charts  
- 🔍 Real-world dataset exploration
